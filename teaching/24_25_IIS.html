Course content:

The course will cover the fundamentals of securing electronic data and information over computer applications, systems, networks, and other relevant technologies. It is designed to equip students with the knowledge of modern information security requirements. A recurring theme for the course shall be security definitions across different digital platforms. Some key topics covered in the course are:

  • Cryptography
  • OS and network security
  • Web security
  • Mobile security
  • ML model security
PS: Course content may vary depending on the interest and response from the students.

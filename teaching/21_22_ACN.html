Course content:

The course will cover the core concepts, as well as the state-of-the-art, of computer networking. It is designed to help the students understand the challenges of establishing a computer network, validate design considerations of networking solutions, and examine the emerging technologies in the field of computer networking. The course is especially useful for students who plan to do research and/or product development in the area of computer networks. Some key topics covered in the course are:

  • Resource allocation
  • Congestion control
  • Queue management
  • Middleboxes
  • Tunneling
  • Interdomain routing
  • Network measurement and traffic engineering
  • DNS and CDN
  • Network security concepts
  • Modern network paradigms: SDN, ICN, CCN, and NDN
PS: Course content may vary depending on the interest and response from the students.

Group project 1:

Each group (group size to be decided) will work on one research paper related to any modern network paradigm (e.g., SDN, ICN, CCN, or NDN). Select a paper published in the last two years from a top computer networking conference. Prepare a LaTeX report (format: single-column, 10pts) and a self-explanatory Beamer presentation (format: any standard template with max 20 slides). The report should contain: (1) a summary of the paper (max 2 pages); (2) major critiques on assumptions, technical approach, analysis, and/or results (no page limit); and (3) suggestions for improvements/extensions (no page limit).

Group project 2:

Each group (group size to be decided) will work on one programming project. Implement a security-oriented and/or privacy-oriented application/feature/protocol (e.g., a firewall) over any modern network paradigm (e.g., SDN, ICN, CCN, or NDN). Submit a single zip file (named as RollNoA_RollNoB_..._RollNoZ.zip) that contains: (1) source code; (2) screenshots of code execution; and (3) clear instructions (in README.txt) to set up environment (if any) and execute the code. No written report is required.
